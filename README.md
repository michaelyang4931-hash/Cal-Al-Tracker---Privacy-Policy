# 隐私政策 (Privacy Policy)

**发布日期：2026年5月11日**

欢迎使用 **Cal AI Tracker**。我们非常重视您的隐私。本隐私政策旨在告知您我们如何收集、使用及保护您的信息。

### 1. 数据收集与使用
我们坚持**数据本地化**原则。您的绝大部分健康数据仅存储在您的设备本地，我们不会在我们的服务器上存储您的个人信息。

* **健康数据 (HealthKit)**：在获得您的明确授权后，我们会读取您的步数和运动数据，以计算每日热量消耗。这些数据仅用于应用内展示，不会被上传或分享。
* **图像数据 (AI 识别)**：当您使用“拍照识食”功能时，图片会通过加密通道发送至第三方 AI 接口（阿里云通义千问 Qwen）进行实时分析。**我们不会存储您的照片**，分析完成后，图片即被销毁。
* **个人中心数据**：您输入的年龄、身高、体重和目标，仅保存在您手机的本地存储（AppStorage）中。

### 2. 第三方服务
* **支付处理**：所有的订阅交易均通过 Apple 的 StoreKit 2 处理。我们不会获取您的银行卡号或支付账号信息。
* **AI 识别引擎**：我们使用阿里云 AI 提供食物识别技术。

### 3. 数据安全
由于我们不要求用户注册账号，所有数据均随 App 存在。如果您卸载 App，本地的所有记录也将随之删除。

### 4. 您的权利
您可以随时在 iOS “设置” -> “隐私” 中撤回对相机或健康数据的访问权限。
